# Voltage Divider Calculator

A simple web-based calculator to compute the output voltage of a voltage divider circuit.

## Formula

Vout = Vin × (R2 / (R1 + R2))

## Features

- Calculates output voltage quickly
- Simple user interface
- Built using HTML, CSS, and JavaScript
- Useful for electronics students and engineers

## How to Use

1. Enter Input Voltage (Vin)
2. Enter Resistor R1 value
3. Enter Resistor R2 value
4. Click **Calculate**

The output voltage will be displayed instantly.

## Technologies Used

- HTML
- CSS
- JavaScript
